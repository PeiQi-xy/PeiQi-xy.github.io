JS预解析

js代码执行过程：1、预解析 2、执行

预解析包含两个方面：1、函数提升 2、变量提升

在函数预解析时，不能跨越script块

​ 

<head>
    <script>
        //函数的提升
        fn1();
        function fn1(){ 
            console.log("1");
        }
        //代码正常运行
        //变量的提升，只提升变量声明，不提升变量赋值
        //console.log("num");
        //var num =10;
        //以上代码可等价为
        var num;
        console.log("num");
        num = 10;
        //函数表达式不会提升，函数表达式的本质就是，把函数当对象使用
    </script>
</head>
